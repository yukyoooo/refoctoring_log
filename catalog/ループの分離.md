# ループの分離 

## 手順
1. ループをコピーする
2. 重複による副作用を特定して排除する
3. テストする

```js
let averageAge = 0;
let totalSalary = 0;
for(const p of people) {
	averageAge += p.age;
	totalSalary += p.salary;
}
averageAge = averageAge / people.length;
```
↓
```js
let totalSalary = 0;
for(const p of people) {
	totalSalary += p.salary;
}

let averageAge = 0;
for(const p of people) {
	averageAge += p.age;
}
averageAge = averageAge / people.length;
```
